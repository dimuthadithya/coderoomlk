6-Month JavaScript Mastery Curriculum

Duration: 6 Months (24 weeks) | Schedule: 2 sessions per week

Session Length: 2 Hours each (4 hours/week) | Total: 96 Hours

🎯 Final Goal: Master JavaScript fundamentals needed for React development

Week 1: Development Setup and Git Fundamentals

  • • Development environment setup with VS Code
  • • Git basics: init, add, commit, push, pull
  • • GitHub repository creation and management
  • • Basic JavaScript syntax and variables

Week 2: Functions and Modern Syntax

  • • Function declarations, expressions, and arrow functions
  • • Template literals and string methods
  • • Let, const, and block scope

Week 3: Arrays and Objects

  • • Array creation, indexing, and basic methods
  • • Object literals, properties, and methods
  • • Destructuring assignment for arrays and objects

Week 4: DOM Basics and Events

  • • DOM selection and manipulation
  • • Event listeners and handling
  • • Project: Interactive Todo List with Local Storage

Week 5: ES6+ Essentials for React

  • • Spread and rest operators
  • • Default parameters and enhanced object literals
  • • Conditional (ternary) operator patterns

Week 6: Async JavaScript Foundation

  • • Callbacks, Promises, and async/await
  • • Error handling with try/catch
  • • setTimeout and setInterval

Week 7: Array Methods Mastery

  • • map(), filter(), reduce() for React lists
  • • find(), some(), every() methods
  • • Method chaining patterns

Week 8: Modules and Organization

  • • ES6 import/export syntax
  • • Module organization best practices
  • • Project: Weather Dashboard with API Integration

Week 9: Closures and Context

  • • Understanding closures and lexical scope
  • • 'this' keyword and binding
  • • Practical closure applications

Week 10: Higher-Order Functions

  • • Functions as first-class citizens
  • • Callback patterns and function composition
  • • Creating reusable utility functions

Week 11: Object-Oriented JavaScript

  • • Classes and inheritance patterns
  • • Prototypes and prototype chain
  • • Encapsulation and abstraction

Week 12: Advanced Objects and State

  • • Object.assign() and deep cloning
  • • State management patterns
  • • Project: E-commerce Product Catalog with Filtering

Week 13: Component Thinking

  • • Breaking UI into components
  • • Component composition patterns
  • • Props-like data passing concepts

Week 14: Event Systems and Patterns

  • • Event delegation and bubbling
  • • Custom event systems
  • • Observer pattern implementation

Week 15: HTTP and API Integration

  • • Fetch API and HTTP methods
  • • JSON handling and data transformation
  • • Error handling in API calls

Week 16: Performance and Optimization

  • • Debouncing and throttling
  • • Memory management best practices
  • • Project: Real-time Chat Interface (WebSocket simulation)

Week 17: Build Tools and NPM

  • • NPM package management
  • • Vite build tool setup and configuration
  • • Development vs production builds

Week 18: Testing Foundations

  • • Unit testing concepts with Jest
  • • Testing functions and modules
  • • Test-driven development basics

Week 19: Advanced Git Workflows

  • • Feature branching and pull requests
  • • Code reviews and collaboration
  • • Merge conflicts and resolution

Week 20: Code Quality and Standards

  • • ESLint and code formatting
  • • Code documentation best practices
  • • Project: Task Management App with Testing

Week 21: Component Architecture Patterns

  • • Container vs presentational components
  • • State lifting patterns
  • • Data flow in component hierarchies

Week 22: Modern JavaScript Patterns for React

  • • Immutability patterns and state updates
  • • Functional programming for React
  • • Advanced array/object manipulation

Week 23: Project Architecture and Deployment

  • • Project structure and organization
  • • Environment variables and configuration
  • • Deployment with GitHub Pages/Netlify

Week 24: Portfolio Development and React Readiness

  • • Creating a professional developer portfolio
  • • React concepts overview and next steps
  • • Final Project: Professional Portfolio Website
  • • Course completion and React readiness assessment
